About the Project

This web application is designed to detect whether a news headline is real or fake using a machine learning model trained on a dataset of labeled news articles.

The goal is to combat the spread of misinformation online by providing a quick and accessible way for users to verify the authenticity of news headlines.

The model is built using Python and trained in Jupyter Notebook, while the frontend and backend are developed using Flask and HTML with Tailwind CSS for styling.

Key Features

  • Detects fake news headlines in real-time
  • Built with Flask web framework
  • Trained using TfidfVectorizer and Machine Learning classifiers
  • Simple and user-friendly UI
